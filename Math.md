# % и // в [python](https://pythonworld.ru/samouchitel-python) ![Alt](Python-logo-notext.svg.png)
## определение
// - выводит ответ без дроби, откидывая ее
% - выводит остаток от деления

## % в коде

```python

a=10
b=5
Print(a % b)

```
Код выведет ответ 0

```python

a=10
b=3
Print(a % b)

```
Код выведет ответ 1


## // в коде

```python

a=10
b=5
Print(a // b)

```
Код выведет ответ 2

```python

a=10
b=3
Print(a // b)

```
Код выведет ответ 3

## Формулы

a % b =

$$
0, 1, 2, ...b-1
%8 = 8 = 8/0
0 < или = a % b < b
$$
