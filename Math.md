## определение
// - выводит ответ без дроби, откидывая ее
% - выводит остаток от деления

## % в коде

```python

a=10
b=5
Print(a % b)

```
Код выведет ответ 0

```python

a=10
b=3
Print(a % b)

```
Код выведет ответ 1


## // в коде

```python

a=10
b=5
Print(a // b)

```
Код выведет ответ 2

```python

a=10
b=3
Print(a // b)

```
Код выведет ответ 3
